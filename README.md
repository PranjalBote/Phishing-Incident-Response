# Phishing Incident Response Playbook
Author: Pranjal Bote

## 🎯 Objectives
- Understand phishing attack stages
- Learn how to detect phishing attempts
- Practice incident response workflow
- Build SOC investigation skills

## 🧠 Topics Covered
- Phishing Life Cycle (Recon → Weaponization → Delivery → Exploitation → C2 → Objective)
- Email Header Analysis
- Indicators of Compromise (IOCs)
- Incident Response Phases
- SOC Playbook & Investigation Checklist

## 🛠 Tools Used
- Nmap
- Wireshark
- Microsoft Defender
- Splunk
- VirusTotal

## 🚀 Goal
To prepare for entry-level SOC Analyst / Cybersecurity roles.
